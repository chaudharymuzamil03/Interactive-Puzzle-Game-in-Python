# 📖 Project Overview

The **Python Sliding Puzzle Game** is an interactive desktop application that recreates the classic sliding puzzle experience. Players must rearrange randomly shuffled numbered tiles by moving one tile at a time until the puzzle reaches its correct order.

The project demonstrates Python programming, GUI development, event-driven programming, and algorithmic thinking while providing an engaging user experience.

# 🎯 Purpose

The objective of this project is to strengthen logical reasoning and problem-solving skills through an interactive puzzle game while demonstrating the implementation of Python programming concepts and graphical user interfaces.

```
Start Game
      │
      ▼
Generate Puzzle
      │
      ▼
Shuffle Tiles
      │
      ▼

      │
      ▼

      │
      ├───────────────┐
      ▼               ▼

      │
      ▼
```
